# AutoPartsManager-WebClient
Web client for the Auto parts management system REST API.

## Used technologies
- HTML;
- CSS;
- JavaScript;
- jQuery;
- Material Design Bootstrap UI kit.

## How to run
1. Clone this repository.
2. Edit the config.js file inside the js folder to point to your Auto parts management system REST API.
3. Have all the files served by a web server (i.e. Apache).

## Notes
- Make sure that the host of the web server that serves the files is added in the allowed origins in the REST API configuration.
